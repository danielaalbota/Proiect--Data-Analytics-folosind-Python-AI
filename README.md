Acest proiect a fost realizat în cadrul cursului „Data Analytics folosind Python & AI” și are ca scop analiza factorilor care influențează performanțele academice ale studenților, precum și construirea unui model de Machine Learning capabil să estimeze nota la examen.
Obiectivele proiectului:
-Realizarea unei analize exploratorii a datelor (EDA) pentru identificarea principalilor factori care influențează performanța academică
-Construirea și antrenarea unui model de Machine Learning pentru estimarea notei finale la examen
-Generarea unui student fictiv și estimarea notei acestuia folosind modelul antrenat
-Crearea, motivarea și testarea unui set de sugestii personalizate pentru îmbunătățirea performanței studentului
Metodologie:
-Curățarea și preprocesarea setului de date
-Analiză statistică și vizualizări pentru evidențierea corelațiilor dintre variabile
-Antrenarea și evaluarea unui model de regresie
-Simularea unui caz real (student fictiv) și interpretarea rezultatelor
-Formularea de recomandări bazate pe impactul variabilelor asupra notei finale
Tehnologii utilizate
-Python
-Pandas, NumPy
-Matplotlib, Seaborn
-Scikit-learn
-Jupyter Notebook

Rezultate:
Modelul obținut oferă o estimare realistă a notei la examen și permite simularea diferitelor scenarii de îmbunătățire a performanței academice prin ajustarea factorilor relevanți.
